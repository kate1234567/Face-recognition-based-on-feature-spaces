# Распознавание лиц на основе пространств признаков

## Описание

Проект представляет собой приложение на Python для распознавания лиц с использованием различных пространств признаков.

В работе реализовано сравнение нескольких методов извлечения признаков и проведена оценка качества распознавания на базе изображений ORL Faces.

Проект выполнен в рамках лабораторной работы по дисциплине, связанной с компьютерным зрением.

## Используемые технологии

* Python
* OpenCV
* NumPy
* Matplotlib
* PyQt5
* Scikit-image

## Реализованные методы

* гистограмма яркости;
* градиентное представление;
* DFT (дискретное преобразование Фурье);
* DCT (дискретное косинусное преобразование);
* масштабирование изображения.

## Возможности

* выбор количества классов;
* выбор количества эталонных изображений;
* сравнение различных методов распознавания;
* визуализация результатов;
* построение графиков точности распознавания.

## Запуск проекта

1. Установить необходимые библиотеки Python.
2. Запустить файл `main.py`.
3. Выбрать количество классов и эталонных изображений.
4. Получить результаты распознавания и графики эффективности методов.
